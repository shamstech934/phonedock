# Install v5.2.0

1. Extract this ZIP over the deployed PhoneDock project or deploy it as the complete project.
2. Keep the existing environment variables unchanged.
3. Run `npm install`.
4. Run `npm run test:v5.2`.
5. Run `npm run typecheck` and `npm run build`.
6. Deploy to Vercel.

The comparison URL remains `/compare?p=slug-1,slug-2` and supports up to six phones.
