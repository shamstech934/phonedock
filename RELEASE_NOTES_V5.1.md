# PhoneDock v5.1.0 — AI Phone Finder

## Added
- Reusable Roman Urdu and English query parser.
- PKR budget formats: `50k`, `1 lakh`, `under 30000`, `60,000 ke andar`.
- Multi-intent detection for gaming, camera, battery, performance, display and value.
- Brand aliases and common spellings including One Plus and Techno/Tecno.
- PTA, AMOLED/OLED, 5G, NFC, RAM, storage, camera, battery and refresh-rate parsing.
- Finder confidence indicator and five-item local search history.
- Regression test file for representative Pakistan-focused searches.

## Database
No migration required.
