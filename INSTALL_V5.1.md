# PhoneDock v5.1.0 Installation

1. Extract this ZIP over the current project or deploy it as a complete replacement.
2. Keep the existing `.env` values.
3. Run `npm install`, then `npm run typecheck` and `npm run build`.
4. Open `/phone-finder` and test the example searches.

No database migration is required for v5.1.0. Recent searches are stored only in the visitor's browser.
