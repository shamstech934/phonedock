# PhoneDock v9.0 PC3 — Repair Work Packs

## Added
- Export the complete Missing Specs, Missing Images, or Missing Prices live queue as CSV.
- Search filters are preserved when exporting the full queue.
- Select individual phones or the entire visible page.
- Export selected rows as a smaller repair work pack.
- Added admin-editor paths to full CSV exports for faster review.
- CSV output includes a UTF-8 BOM for reliable Excel/Urdu compatibility.

## Safety
- Exports are read-only and never invent, auto-approve, or publish specifications, images, or prices.
- Imported repair data should still be reviewed by an administrator before publishing.
