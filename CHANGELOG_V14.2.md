# PhoneDock v14.2.0

- Added visible **Find specs** button inside Admin → Data Quality → Missing Specs.
- Added free community API search with server-side timeout and legacy/v2 fallback.
- Added candidate match confidence and preview/edit modal.
- Added explicit admin approval before MongoDB update.
- Added source metadata, verification timestamp, numeric filter extraction, and activity logging.
- No AI provider, paid credits, or automatic database writes.

Optional environment variable:

`FREE_PHONE_SPECS_API_BASE=https://api-mobilespecs.azharimm.dev`

The default is already built in, so the variable is normally not required.
