# Installation

1. Extract this ZIP.
2. Copy the included `src` folder into the root of your PhoneDock repository.
3. Allow it to overwrite the existing file:
   `src/app/api/[[...path]]/handlers/admin-crud.ts`
4. Commit and push the changed file to GitHub.
5. After Vercel deploys, open **Admin → Reviews** and click **Retry** or refresh the page.

No package installation, environment variable, or database migration is required.
