# PhoneDock Production Launch Fix Pack

## Completed in this package

- Expanded Hero Banner CMS controls.
- Added editable CTA button text and URLs to the live homepage hero.
- Added hero phone animation enable/disable control.
- Added animation speed control from 2 to 12 seconds.
- Added show/hide control for the animated phone information card.
- Added a visual hero preview inside Admin Site Settings.
- Fixed Quick View data detection so it no longer requires the display field before showing available specifications.
- Corrected the homepage Trending section link to open the trending collection.
- Added phone specification sharing for WhatsApp, Facebook, X, Telegram, email, native device sharing, and copy link.

## Documentation policy

All new project Markdown documentation should be stored inside the `/docs` directory rather than the project root.

## Deployment notes

No new environment variables or database migrations are required. Existing settings documents remain compatible; new hero settings use safe defaults.
