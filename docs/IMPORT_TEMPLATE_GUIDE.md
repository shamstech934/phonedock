# Import template guide

Phone templates require `brand` and `model`. Dates use ISO `YYYY-MM-DD`; prices are numeric PKR; RAM/storage/battery/charging include units (`8 GB`, `256 GB`, `5000 mAh`, `67 W`); booleans accept true/false, yes/no or 1/0. URLs must be public HTTPS locations with a valid usage basis. Unknown values must be blank, never guessed.

Price templates require brand, model, source/store, currency, price, collection timestamp and source record ID. Expiry, warranty, PTA status and affiliate URL are optional. Prices are reviewed independently and low-confidence values never replace the canonical Pakistan price automatically.
