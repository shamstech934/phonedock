# PhoneDock Documentation

This folder is the single location for project documentation.

## Current documents

- `FULL_PRODUCTION_AUDIT_2026-07-21.md` — current combined production audit.
- `INSTALL.md` — installation and deployment notes.
- `PATCH_NOTES.md` — latest patch notes.
- `Project-Size-Audit.md` — project-size and generated-file guidance.
- `ADMIN_REVIEWS_404_FIX.md` — Reviews API 404 correction.

## Archive

Historical sprint and audit reports are retained under `docs/archive/legacy-audits/`. They are not required at runtime and may be removed from a deployment-only package.

Project documentation must not be placed in `download/` or scattered in the repository root. Root should contain only source-control, build, and package configuration files.
