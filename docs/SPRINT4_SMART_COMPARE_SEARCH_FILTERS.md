# Sprint 4 — Smart Compare, Search and Filters

## Included

- Compare is now consistently limited to four phones across the UI and lookup API.
- A native share / copy-link action was added to the comparison toolbar.
- Public phone search now also matches brand names and selected specification fields such as chipset, CPU, GPU, RAM, storage, display type and operating system.
- New URL-persistent filters were added for display technology, refresh rate, main camera resolution and battery capacity.
- Existing pagination, sorting and filter reset behavior is preserved.

## Deployment

Copy the `src` and `docs` folders into the repository root and allow overwrite. No dependency, environment variable or database migration is required.
